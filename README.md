# 📊 Vrinda Store Annual Report Dashboard (Excel)

An interactive Excel dashboard built to analyze the annual sales performance of **Vrinda Store**. This dashboard provides business insights using Pivot Tables, Pivot Charts, Slicers, and Excel data visualization features.

The dashboard is fully interactive using slicers for quick filtering.

## 📊 Dashboard Insights

### Orders vs Sales
- Monthly comparison of total sales and number of orders.

### Orders by Channel
- Amazon contributes the highest sales.
- Myntra and Flipkart are also major contributors.

### Order Status
- Over **92%** of orders were successfully delivered.
- Very low cancellation and return rate.

### Top States
Highest sales were generated from:
- Maharashtra
- Karnataka
- Uttar Pradesh
- Telangana
- Tamil Nadu

### Age vs Gender
- Adult women contribute the highest percentage of sales.
- Women's purchases are significantly higher than men's.

### Sales by Gender
- 👩 Women: **64%**
- 👨 Men: **36%**

## 🚀 How to Use

1. Download the Excel file.
2. Open it in Microsoft Excel (2019 or Microsoft 365 recommended).
3. Use the slicers on the left side to filter:
   - Month
   - Category
   - Sales Channel
4. Explore the dashboard interactively.
